Esta documentación se encontraran los prompts y paso a paso de la creacion de esta app.

  1. Petición inicial: Actuar como experto en HTML/CSS/JS puro y crear una aplicación tipo "guía" (sobre un pueblo, películas, etc.) con modo oscuro, tarjetas y transiciones.

  2. Definir contenido: Concretar la información: Guía sobre Pinto y una lista de 10 películas favoritas (tipo Alien, Expediente Warren).

  3. Diseño dos columnas: Cambiar la estructura a dos columnas (Pinto vs Películas), añadir degradados direccionales específicos (flechas negra y verde) y efecto de movimiento en la imagen al pasar el ratón.

  4. Ajuste de tarjetas: Hacer las columnas más altas, imágenes más grandes que cubran el fondo y evitar espacios blancos al hacer hover. Aqui ya pasé de hacer una guia de pinto y me centre en las peliculas

  5. Detalle completo (El gran cambio): Pedir que se muestren las puntuaciones, estrellas, sinopsis y opiniones con un diseño de tarjeta detallada (basado en tu captura), en lugar de la lista de texto simple que teníamos.

  6. Enriquecimiento de datos y expansión de contenido "Utilizando la nueva API (OMDb), amplía la base de datos local añadiendo más categorías de género y películas en cada una. Además, implementa una llamada a la API para obtener descripciones (sinopsis) más detalladas dinámicamente, manteniendo la estructura de código unificado."
  7. Enriquecimiento de datos y expansión de contenido "Utilizando la nueva API (OMDb), amplía la base de datos local añadiendo más categorías de género y películas en cada una. Además, implementa una llamada a la API para obtener descripciones (sinopsis) más detalladas dinámicamente, manteniendo la estructura de código unificado."
  8. Los demas prompts son arreglos visuales o de bugs y hacerlo multiplataforma
