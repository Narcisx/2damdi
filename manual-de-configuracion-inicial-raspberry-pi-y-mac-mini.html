Sigue estos pasos para dejar listos los equipos Raspberry Pi y Mac mini.

 

Configuración de Raspberry Pi

 

  1. Pantalla de Bienvenida

    Verás el escritorio de la Pi. Haz clic en 'Next'. Nota: Si el sistema necesita configurar el Bluetooth, se reiniciará solo.

     

  2. Elegir WiFi

    Busca la red 'WiFi_AULA_ATECA' en la lista y continúa.

  3. Clave WiFi

     Escribe la contraseña de la red seleccionada y pulsa 'Next'.

  4. Actualizar Sistema

    Aquí puedes actualizar el software pulsando 'Next' o saltarlo con 'Skip'. Lo mas recomendable es actualizarlo.

  5. Nuevo Usuario

    Define el nombre de usuario y la contraseña que usarás en la Raspberry.

  6. Navegador Web

    Elige Firefox como tu navegador por defecto. Puedes marcar la casilla para desinstalar Chromium si no vas a usarlo.

 

Configuración de Mac mini

  1. Crear Cuenta

    Rellena los datos para el usuario local (nombre real, nombre de cuenta y contraseña).

  2. Listo para usar

    Una vez creada la cuenta, accederás directamente al escritorio de macOS.
